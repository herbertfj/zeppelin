Welcome to the Archimedes Tool Box!

Archimedes is our big data engine storing thousands of data series on over 300,000 global stocks. This toolbox is web-based notebook that enables interactive data analytics.
You can make beautiful data-driven, interactive, collaborative document with SQL, code and even more! We built this toolbox on top of Zeppelin, an open-source project in the Apache Spark community.
